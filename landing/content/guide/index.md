This guide walks you through setting up your own AI Cabinet — a team of specialized AI agents that handle operations, engineering, content, strategy, coaching, and community work for your solo founder journey.

## What You'll Set Up

By the end of these guides, you'll have:

- **Claude Code** installed and configured as your agent runtime
- **Your first agent** (Maya, Chief of Staff) running daily reviews
- **Telegram bots** (optional) for direct messaging with each agent
- **Scheduled tasks** for daily and weekly review cadences
- **S3 governance** so each agent knows its boundaries

## Guides

### 1. [Installing Claude Code](/guide/install-claude-code)
Get the AI runtime installed, authenticated, and understand the key concepts behind agent skills and configuration.

### 2. [Setting Up Your First Agent](/guide/first-agent)
Create Maya (Chief of Staff) as your foundation agent. She handles the daily rhythm that everything else builds on.

### 3. [Configuring Telegram Bots](/guide/telegram-bots)
Optional but recommended. Give each agent its own Telegram bot so you can message them directly from your phone.

### 4. [Daily Reviews & Scheduled Tasks](/guide/daily-reviews)
Set up the recurring rhythms — daily check-ins, weekly planning, and automated inbox processing.

### 5. [S3 Governance for Agent Teams](/guide/s3-governance)
Define clear ownership domains for each agent using Sociocracy 3.0 patterns. This prevents overlap and ensures accountability.

## Prerequisites

- **Node.js 18+** installed on your machine
- **An Anthropic API key** (sign up at anthropic.com)
- **A terminal** you're comfortable working in
- **30 minutes** for initial setup

## Philosophy

The AI Cabinet is opinionated about a few things:

- **Agents need boundaries.** Each agent has a clear domain, defined constraints, and knows what it cannot do without your approval.
- **You are the commander.** Agents propose, you decide. No agent acts autonomously on irreversible decisions.
- **Start small, expand gradually.** Begin with one agent (Maya). Add others only when you feel the gap.
- **Skills are composable.** Each skill is a standalone capability that can be mixed and matched across agents.
