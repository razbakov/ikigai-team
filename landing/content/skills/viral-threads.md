# Viral Threads

This skill takes a blog post, article, talk transcript, or any long-form content and distills it into a punchy social media thread. It extracts the most compelling insights, structures them for engagement, and formats them for platform-specific requirements.

A great thread does three things: hooks attention in the first line, delivers value in every post, and ends with a clear call to action. This skill handles the structure — you provide the substance.

## How It Works

1. **Analyze the source** — Read the input content and identify the core narrative, key insights, surprising facts, and actionable takeaways.
2. **Extract thread hooks** — Find the single most compelling angle for the thread opener. The first line determines whether anyone reads the rest.
3. **Structure the thread** — Organize insights into a logical flow: hook, context, insights (one per post), and conclusion with CTA. Aim for 5-12 posts.
4. **Format for platform** — Apply platform-specific constraints: character limits, line breaks, emoji usage, and hashtag strategy. Different platforms need different formatting.
5. **Add engagement elements** — Include questions, polls, or prompts that invite replies. Engagement signals boost algorithmic distribution.

## When to Use

- After publishing a blog post — repurpose it as a thread for social reach
- When you have a talk, podcast, or interview worth sharing
- To share lessons learned from a project or experiment
- When you want to test an idea with your audience before writing a full post

## Requirements

- Source content to transform (blog post, notes, transcript, or key ideas)
- Brand voice guidelines in your `CLAUDE.md` so threads match your style
- Knowledge of your target platform (the agent adapts formatting accordingly)
- No integrations required — the output is text you can post anywhere
