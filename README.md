# Ikigai Team

Your AI dream team. 6 agents with Level 10 Life, GROW coaching, OKRs, GTD, and S3 governance built in. Starts with a coaching session, not a config file.

## The Team

| Agent | Role | Personality |
|-------|------|------------|
| **Sage** | Personal Coach | INFJ — insightful, empathetic, purposeful |
| **Marco** | Strategy & Business | ENTJ — analytical, decisive, results-focused |
| **Maya** | Chief of Staff | ISTJ — reliable, systematic, detail-oriented |
| **Viktor** | CTO | INTJ — strategic, efficient, quality-focused |
| **Luna** | Content & Growth | ENFP — imaginative, enthusiastic, trend-aware |
| **Kai** | Community & Partnerships | ESFJ — warm, social, connector-minded |

## Install

```bash
cp -r skills/setup-ikigai ~/.claude/skills/setup-ikigai
```

Then start a new Claude Code conversation and say:

```
Set up my Ikigai Team
```

The journey:
1. **Sage** helps you reflect — Level 10 Life assessment, find your direction
2. **Marco** turns your direction into OKRs — measurable quarterly goals
3. **Maya** sets up the daily rhythm — GTD inbox processing, reviews, planning
4. **The team** handles the rest — code, content, community. You focus on your ikigai.
